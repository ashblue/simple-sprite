Simple Sprite

Turn an image into a scaled sprite sheet with just one line of code. Accepts images of data:uri, <img>, and url paths.

new SimpleSprite('/images/vulture.png', 17, 13, {
    scale: 4,
    target: 'vulture',
    frame: 0,
    sequence: [0, 1, 2, 3, 4, 5, 4, 5, 4, 5, 4, 5],
    repeat: true,
    speed: 0.3,
    flipX: false
});
